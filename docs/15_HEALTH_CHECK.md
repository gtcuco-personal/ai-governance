# Weekly Health Check

> Run once per week.

## 1. Content & Structure

- [ ] No stale or outdated content
- [ ] All links and references are valid

## 2. Repository Hygiene

- [ ] No uncommitted changes in main
- [ ] No sensitive data committed
- [ ] `CHANGELOG.md` includes recent changes

## 3. Roadmap

- [ ] `docs/5_ROADMAP_AND_TASKS.md` has no stale in-progress items
- [ ] Last updated date is current

---

## How to use

1. Run every Monday
2. Log action items in `docs/5_ROADMAP_AND_TASKS.md`
3. Adapt as the repo evolves
